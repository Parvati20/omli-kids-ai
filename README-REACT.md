# Omli Kids - AI Learning Buddy (React Version)

NVIDIA API key ko `.env.local` file mein dalo:

```
NEXT_PUBLIC_NVIDIA_API_KEY=your_nvidia_api_key_here
```

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Browser mein kholo: http://localhost:3000

## Features

✅ Speech-to-Speech library se TTS
✅ Browser SpeechRecognition se STT
✅ NVIDIA Nemotron AI integration
✅ Kid-friendly UI (pink theme)
✅ Real-time chat interface
